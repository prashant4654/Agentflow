HOW TO USE
=============================================================================

Basic Configuration:
  from agentflow.utils import configure_logging
  import logging
  configure_logging(level=logging.DEBUG)

Quick Debug Mode:
  from agentflow.utils import enable_debug
  enable_debug()

Environment Variables:
  export AGENTFLOW_DEBUG=1
  export AGENTFLOW_LOG_REASONING=1

Reasoning Capture:
  # Automatically logged when AGENTFLOW_LOG_REASONING=1
  # Shows in format: [REASONING] Node=x, Message=y, Block=z: summary
